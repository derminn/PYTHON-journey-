# TUPLES

## Introduction to Tuples

Tuples are **immutable** sequences in Python, meaning once created, their elements cannot be changed, added, or removed. They are defined using parentheses `()`.

### Why Use Tuples?

- **Immutability**: Data integrity - values can't be accidentally modified
- **Performance**: Tuples are faster than lists
- **Memory**: Tuples use less memory than lists
- **Dictionary Keys**: Tuples can be used as dictionary keys (lists cannot)
- **Function Returns**: Perfect for returning multiple values from functions

---

## Looping Through Tuples

### 1. Basic For Loop

The simplest way to iterate through a tuple:

python

`colors = ("red", "blue", "pink")
for color in colors:
    print(color)`

**Output:**

`red
blue
pink`

---

### 2. Loop Using Index

When you need both the index and the value:

python

`colors = ("red", "blue", "pink")

for i in range(len(colors)):
    print(i, colors[i])`

**Output:**

`0 red
1 blue
2 pink`

---

### 3. Access Index + Value Together (Formatted)

Using f-strings for better formatting:

python

`fruits = ("apple", "banana", "mango")

for i in range(len(fruits)):
    print(f"index {i} -- {fruits[i]}")`

**Output:**

`index 0 -- apple
index 1 -- banana
index 2 -- mango`

---

### 4. Using Index for Conditional Logic

Filter elements based on index position:

python

`data = (5, 10, 15, 20, 25)

for i in range(len(data)):
    if i % 2 == 0:  # Even indices only
        print(data[i])`

**Output:**

`5
15
25`

**Use Case**: Skip every other element, process alternating items, etc.

---

### 5. Comparing Neighboring Elements

Compare consecutive elements in a tuple:

python

`nums = (3, 8, 2, 9)

for i in range(len(nums) - 1):
    if nums[i] < nums[i + 1]:
        print(nums[i], "is smaller than", nums[i + 1])`

**Output:**

`3 is smaller than 8
2 is smaller than 9`

**Important**: Use `range(len(nums) - 1)` to avoid `IndexError` when accessing `nums[i + 1]`

---

### 6. Reverse Looping Using Index

Loop through a tuple in reverse order:

python

`letters = ("a", "b", "c", "d")
for i in range(len(letters) - 1, -1, -1):
    print(i, letters[i])`

**Output:**

`3 d
2 c
1 b
0 a`

**Syntax**: `range(start, stop, step)`

- `start`: `len(letters) - 1` (last index)
- `stop`: `1` (goes until 0, inclusive)
- `step`: `1` (decrement)

---

### 7. Simple Transformation Loop

Apply operations to each element:

python

`t = (1, 2, 3)
for i in range(len(t)):
    print(t[i] * 2)`

**Output:**

`2
4
6`

---

### 8. Nested Loops with Index

Work with tuples containing tuples:

python

`pairs = ((1, 2), (3, 4), (5, 6))

for i in range(len(pairs)):
    print("Pair", i, "=", pairs[i])`

**Output:**

`Pair 0 = (1, 2)
Pair 1 = (3, 4)
Pair 2 = (5, 6)`

---

### 9. Using `enumerate()` (Recommended!)

The **Pythonic way** to get both index and value:

python

`names = ("dermin", "derminotsocool", "dermicool")
for index, value in enumerate(names):
    print(index, "=", value)`

**Output:**

`0 = dermin
1 = derminotsocool
2 = dermicool`

**Why use enumerate?**

- More readable than `range(len())`
- Cleaner syntax
- Pythonic and preferred by Python community

## While Loops with Tuples

### 1. Reverse Looping with While

python

`t = (1, 2, 3, 4, 5)

i = len(t) - 1
while i >= 0:
    print(t[i])
    i -= 1`

**Output:**

`5
4
3
2
1`

---

### 2. While Loop with Index + Value

python

`t = ("apple", "banana", "mango")

i = 0
while i < len(t):
    print("index", i, "value", t[i])
    i += 1`

**Output:**

`index 0 value apple
index 1 value banana
index 2 value mango`

---

### 3. While Loop with Conditional Logic

python

`t = (1, 2, 3, 4, 5, 6, 7)

i = 0 
while i < len(t):
    if t[i] % 2 == 0:
        print(t[i], "is even")
    else:
        print(t[i], "is odd")
    i += 1  # Note: Your code had i+=0 which would cause infinite loop!`

**Output:**

`1 is odd
2 is even
3 is odd
4 is even
5 is odd
6 is even
7 is odd`

## Tuple Unpacking

### 1. Unpacking Inside Loop

Directly unpack nested tuples:

python

`data = ((1, 2), (3, 4), (5, 6))

for a, b in data:
    print("a =", a, "b =", b)`

**Output:**

`a = 1 b = 2
a = 3 b = 4
a = 5 b = 6`

---

### 2. Looping Through Mixed Data Types

python

`info = ("dermin", 21, True, 9.5)

for item in info:
    print(item, type(item))`

**Output:**

`dermin <class 'str'>
21 <class 'int'>
True <class 'bool'>
9.5 <class 'float'>`

---

### 3. Tuple Packing

Creating a tuple without parentheses:

python

`t = 10, 20, 30  # Tuple packing
print(t)  # (10, 20, 30)`

---

### 4. Tuple Unpacking

Extracting values from a tuple:

python

`t = (10, 20, 30)
a, b, c = t  # Tuple unpacking
print(a, b, c)  # 10 20 30`

---

### 5. Swapping Values (Pythonic Way!)

python

`x = 5
y = 20

x, y = y, x  # Swap without temporary variable!

print(x)  # 20
print(y)  # 5`

**How it works**: Python evaluates the right side first, creating a tuple `(20, 5)`, then unpacks it to `x` and `y`.

---

### 6. Returning Multiple Values from Functions

python

`def calc(a, b):
    return a + b, a - b  # Returns a tuple

result = calc(10, 5)
print(result)  # (15, 5)

# Or unpack directly:
sum_result, diff_result = calc(10, 5)
print(sum_result)   # 15
print(diff_result)  # 5`

## Tuple Methods

Tuples have only **2 methods** due to their immutability:

### 1. `count()` - Count Occurrences

python

`t = (1, 2, 3, 4, 4, 5, 6, 6, 7, 7, 8)

print(t.count(4))  # 2 (4 appears twice)
print(t.count(7))  # 2
print(t.count(1))  # 1`

---

### 2. `index()` - Find First Occurrence

python

`t = (1, 2, 3, 4, 4, 5, 6, 6, 7, 7, 8)

print(t.index(4))  # 3 (first occurrence at index 3)
print(t.index(7))  # 8 (first occurrence at index 8)`

**Note**: Raises `ValueError` if element not found.

---

## Tuple Comprehension

There's no direct "tuple comprehension", but you can use a generator expression:

python

`t = tuple(x * x for x in range(5))
print(t)  # (0, 1, 4, 9, 16)`

**Syntax**: `tuple(expression for item in iterable)`

More examples:

python

`# Even numbers
evens = tuple(x for x in range(10) if x % 2 == 0)
print(evens)  # (0, 2, 4, 6, 8)

# String lengths
words = ("apple", "banana", "cherry")
lengths = tuple(len(word) for word in words)
print(lengths)  # (5, 6, 6)`

## Tuple Immutability

### The Core Concept

python

`t = (1, 2, 3)
# t[0] = 10  # TypeError: 'tuple' object does not support item assignment`

---

### The Nuance: Mutable Objects Inside Tuples

**Important**: While tuples themselves are immutable, they can contain mutable objects (like lists), and those objects **can** be modified!

python

`t = (1, [2, 3], 4)
t[1].append(5)  # This works! Modifying the list inside

print(t)  # (1, [2, 3, 5], 4)`

**Why this works**:

- The tuple still points to the same list object (reference unchanged)
- The list itself is mutable, so its contents can change
- The tuple's immutability means you can't reassign `t[1]` to a different object

python

`t = (1, [2, 3], 4)
# t[1] = [10, 20]  # This would raise TypeError!`

## Converting Between List and Tuple

### List → Tuple → Modified List → Tuple

python

`t = (1, 2, 3)

# Convert to list
lst = list(t)
lst.append(4)  # Modify the list

# Convert back to tuple
t = tuple(lst)
print(t)  # (1, 2, 3, 4)`

**Use Case**: When you need to modify a tuple, convert it to a list, modify, then convert back.

## Summary: Tuples vs Lists

| Feature | Tuple | List |
| --- | --- | --- |
| Syntax | `(1, 2, 3)` | `[1, 2, 3]` |
| Mutability | Immutable | Mutable |
| Performance | Faster | Slower |
| Memory | Less | More |
| Methods | 2 (`count`, `index`) | Many (`append`, `remove`, etc.) |
| Use Case | Fixed data, returns | Dynamic data |

## Common Pitfalls

### 1. Single Element Tuple

python

`# Wrong
t = (5)  # This is just an int!
print(type(t))  # <class 'int'>

# Correct
t = (5,)  # Note the comma!
print(type(t))  # <class 'tuple'>`

---

### 2. Infinite Loop in While

python

`# Bug in your code:
i = 0
while i < len(t):
    print(t[i])
    i += 0  # This should be i += 1 !!`

---

### 3. Index Out of Range

python

`t = (1, 2, 3)
# for i in range(len(t)):
#     print(t[i + 1])  # Error on last iteration!

# Correct:
for i in range(len(t) - 1):
    print(t[i], t[i + 1])`
